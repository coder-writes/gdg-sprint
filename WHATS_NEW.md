## ğŸ”§ Setup Instructions

### Prerequisites
- Node.js 18+
- MongoDB
- Google Gemini API Key

### Quick Start

1. **Get Gemini API Key**
   - Visit: https://makersuite.google.com/app/apikey
   - Create API key

2. **Configure Backend**
   ```bash
   cd backend
   cp .env.example .env
   # Add GEMINI_API_KEY to .env
   npm install
   npm run dev
   ```

3. **Configure Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Access AI Tools**
   - Open http://localhost:5173
   - Login/Register
   - Navigate to `/ai-tools`
